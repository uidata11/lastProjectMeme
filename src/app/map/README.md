# Map 페이지

1. 대전 내에서 키워드를 검색창에 입력하면 리스트가 나옴
2. 리스트 버튼을 누르거나 지도의 마커를 클릭하면 해당 장소의 한 가운데로 이동 후 상세 정보창이 나옴
3. 상세정보창과 리스트에는 해당 장소의 대략적인 주소 및 전화번호가 있음
4. 상세정보창의 버튼을 누르면 각각 장소의 주소와 전화번호가 복사됨
